This is my work with java for the year 2024 .  Making a ping pong and  number guess game in Java helped me learn a lot about programming. I figured out how to create a game window using `JFrame` and draw the paddles and ball with `Graphics`. I also learned how to let players move the paddles by pressing keys using the `KeyListener`. I wrote code to make the ball bounce off the paddles and walls by checking where they touch. I used a game loop to keep everything running smoothly. This project helped me improve my coding and problem-solving skills, and I now understand how games work better.
